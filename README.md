# money-compass 🧭

一个可以给 AI 助手加载的**金融分析 & 理财规划技能**——把任何一个基金/股票/理财产品"想清楚"的固定流程清单。

> ⚠️ **免责声明**:本项目是**学习用的思考框架**,**不是投资建议**,不推荐任何具体标的、不预测涨跌。真金白银的决定请自行负责,大额或复杂产品请咨询持牌专业人士。

## 这是什么

它把一套"防偷懒、防瞎编"的分析纪律写成了一个标准的 `SKILL.md`:
- **六条原则**:凡数字必带来源+日期、时效内容当场现查、标可信度、强制找反方、收尾说"我可能错在哪"、查不到就说查不到。
- **六步分析流程** + **四道门筛选** + **0-10 速览卡**。
- **进攻评分算法**:算赔率(不是预测价格),带"追高温度计"。
- **理财规划模板**:目标/期限、风险承受力、资产配置、应急金、定投与再平衡、机动仓纪律。
- **信源可信度分级**:不用懂行业,也能识别假消息和骗局。

## 怎么用(Codex)

`SKILL.md` 是 Codex 的标准技能格式。两种装法:

**A. 项目级(推荐,跟着仓库走)**
把本仓库里的 `.agents/skills/financial-analysis/` 文件夹,放进你自己项目的根目录。Codex 会自动扫描 `.agents/skills/` 并识别这个技能。

**B. 全局级(任何项目都能用)**
```bash
git clone https://github.com/unaya1102/money-compass.git
mkdir -p ~/.codex/skills
cp -r money-compass/.agents/skills/financial-analysis ~/.codex/skills/
```
之后在 Codex 里直接说「帮我分析一下〈某基金/某股票〉」即可触发。

## 也兼容其他工具

`SKILL.md` 在 2026 年已成为跨工具开放标准,Claude Code、Gemini CLI、Cursor、GitHub Copilot 等同样可加载同一个文件。
